SponsorR3m.com Redesign

This was my final portfolio piece for the Front-End Development course I took at Betamore. I chose to do a website redesign for my non-profit sponsor a child program, which had previously been designed through the tools of Squarespace.

This site features the use of media queries, fluid grid system from scratch, JavaScript, and advanced CSS.

Responsive Page with Media Qureries and JavaScript

This page demonstrates my ability to make a responsive page through the use of media queries. In keeping with the "Mobile First" school of thought, this was designed to work first on mobile, then adjust once the screen gets larger than 768px.

The navigation bar also features Javascript that puts a border around the active page.

Fluid Grid Framework

Responsive HTML Email Templates
